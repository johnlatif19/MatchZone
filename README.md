# MatchZone - Football Management System

A complete football management platform built with Express.js, Firebase, and JWT authentication.

## 🌐 Live Demo
[https://matchzzone.vercel.app](https://matchzzone.vercel.app)

## 🚀 Features

- **Authentication**: JWT-based secure admin login
- **Team Management**: Create, read, update, delete teams with logo upload
- **Player Management**: Add players with photos, track goals and assists
- **Match Management**: Schedule matches, update scores, track status
- **Tournaments**: Create tournaments and manage brackets
- **Dashboard**: Real-time statistics with charts
- **Image Upload**: Firebase Storage integration
- **Production Ready**: Optimized for Vercel deployment

## 🛠️ Tech Stack

- **Backend**: Node.js, Express.js
- **Authentication**: JWT (JSON Web Tokens)
- **Database**: Firebase Firestore
- **Storage**: Firebase Storage
- **Deployment**: Vercel
- **File Upload**: Multer
